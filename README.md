# Fake News Detection System

This project implements a **Fake News Detection System** using **NLP and Machine Learning**.

## Features

- Classifies social media posts as **Real** or **Fake**
- Built with **Logistic Regression** and **TF-IDF**
- Streamlit frontend for interactive testing
- Evaluation metrics: Accuracy, Precision, Recall, F1 Score

## Files

- `Fake_News_Detection.ipynb` – Colab notebook with data preprocessing, model training, and evaluation
- `app.py` – Streamlit interactive app
- `model.pkl` – Trained ML model
- `vectorizer.pkl` – TF-IDF vectorizer
